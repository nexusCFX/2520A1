To compile:
Open a terminal and navigate to the folder containing bchester_a3.cob
Execute the following command:
cobc -x -free -Wall a3_bchester.cob -o TextAnalyzer

To run the program:
Assuming it has been compiled, it can be run by typing ./TextAnalyer
After doing this, you will be prompted to enter a filename. Enter the name
of the file you want to analyze. When the program is complete, it will output
its analysis results to a file named TextAnalysis.txt, and then exit.
